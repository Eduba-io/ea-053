# CampusRun landing page

A one-page site built for Lili Demm and Jack Sardina at CampusRun (booth SS66 at eMerge Americas). It carries the CampusRun voice and cadence (Poster posts task, Runner finds task, done) with an honest read on where the business should spend time next and a clear CTA to a 30-minute call with Matt Creamer, CRO at Eduba.

Intended URL path: /for/campusrun

## Files

- `index.html`, static HTML, no JavaScript
- `styles.css`, all styling, mobile-first, uses only CampusRun brand tokens (black, white, yellow, warm neutral) and Futura / Helvetica Neue font stacks from their live site
- `logo.png`, CampusRun wordmark copied from Brand Pack/assets/

## Notes

- Pitch body and hook come from `Companies/CampusRun/Research Docs/fit-analysis.md` (sections 5 and 7).
- CTA points to `https://calendly.com/thecro-eduba/30min`.
- No Eduba logo or colors at the top. Eduba appears only in the footer as "Prepared by Eduba, eduba.io" per template disclosure rules.
- No pricing disclosed: the Feeld engagement is referenced with scope language only ("scoped sprint").
- NLP Logix mention includes the mandatory credibility phrasing ("in machine learning since 2011" and "over 150 data scientists").
- All external URLs are real anchors with `target="_blank"` and `rel="noopener"`.
